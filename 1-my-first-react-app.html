C’est quoi React ?

C’est une bibliothèque JavaScript pour créer des interfaces utilisateurs.
Développée par Facebook et publiée en 2013
Permet d’écrire des composants autonomes qui possedent leur propre état.
Lorsque les données changeront, React mettra à jour, de façon optimale, uniquement les composants qui en auront besoin.

React n’est pas un framework, nous avons donc le choix de mettre en place les libraries que nous voulons.
Typage avec Typescript, PropTypes ou rien.
AJAX avec fetch, axios, ...
Routing avec react-router, reach-router, ...
etc...

On peut décider de créer une SPA (Single Page Application) à partir de zéro ou bien progressivement venir remplacer certains parties d'une application existante par des composants React.

React est un langage déclaratif, c'est à dire que l'on ne va pas manipuler directement le DOM (Document Object Model). À l’opposé du langage impératif (par exemple avec jQuery) où l’on va devoir dire au navigateur ce qu‘il faut faire pour manipuler le DOM.

Bonus : React permet de faire des applications web mais également des applications mobile native avec React Native.

Les composants

    
Module 1 : Composant simple et composant à état

    
Composant simple (Stateless component)

    

      Les composants React implémentent une méthode render() qui prend des
      données en entrée et retourne ce qui doit être affiché. Cet exemple
      utilise une syntaxe qui ressemble à du XML et qu’on appelle JSX. Les
      données passées au composant sont accessibles dans render() via
      this.props.
      

    

    
Composant à état (Statefull component)

    

      En plus de pouvoir recevoir des données (accessibles via this.props), un
      composant peut maintenir un état local (accessible via this.state).
      Lorsque l’état local d’un composant change, son affichage est mis à jour
      en appelant render() une nouvelle fois.
    

    
Module 2 : Fonctions composants et composants à base de classes

    
Fonction composant (Functional Component)

    

      C’est un composant React qui accepte un seul argument « props » («
      propriétés ») contenant des données, et renvoie un élément React.
    

    
Composant à base de classe (Class Component)

    

      C'est un composant React écrit sous forme de classe qui possède un état et
      un cycle de vie.
    

    
Module 3 : Cycle de vie

    

      La méthode componentDidMount() est exécutée après que la sortie du
      composant a été injectée dans le DOM.

      Lorsque le composant finit par être retiré du DOM, React appellera la
      méthode de cycle de vie componentWillUnmount().
