Déployer une application React

Création de l’application de production

yarn build

Lance la commande react-scripts build (définie dans le fichier package.json) et crée une version minifiée prête pour la production.

Déploiement sur un serveur local ou hébergé

Avec Node

npm install -g serve
serve -s build

Avec NGINX

server {
  listen 80;
  root /var/www/my-app/build;
  server_name mydomain.com www.mydomain.com;
  index index.html;
  location / {
  }
}
